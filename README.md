# Breast Cancer Prediction App

[![Python](https://img.shields.io/badge/python-3.11-blue)](https://www.python.org/)
[![Flask](https://img.shields.io/badge/Flask-2.3.2-green)](https://flask.palletsprojects.com/)
[![License](https://img.shields.io/badge/license-MIT-orange)](LICENSE)

A **Machine Learning web application** to predict whether a breast tumor is **benign** or **malignant** using patient data. Built with **Python**, **scikit-learn**, and **Flask**.
## Features

- Predict breast cancer with high accuracy.
- Simple and interactive web interface.
- Easy to extend and retrain with new datasets.
